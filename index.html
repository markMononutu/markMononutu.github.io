This page contains my portfolio, both my own work and the results of team work with my colleagues. The existing projects I worked on while I was studying at the Universitas Klabat and also after I finished my studies.

Portfolio

Yukey Resto App (2021)

This is my first project using React Native as part of a mobile application development class. As a back-end service we used Firebase. The idea of the project is to create an application that can help the service in a restaurant especially in ordering food process. This simple project was done in a team consisting of: Gracia Palangan, Ferdinand Rattu, Gillian Saroinsong, Claudio Mambu and me.



Surat Cinta BO (2021)

Surat Cinta BO is a running Video Game. This project was developed using the Unity engine and the C# programming language. This project is part of the final assignment for Computer Graphics class. Our team members are Revdian Paseki, Tarisha Hamzah and me.



KiosNote App (2021)

This is a simple project as part of the Back-End Development Class Project. The idea of KiosNote is to record all the sells of a stall, and can be saved to the database. This project created with the ReactJS library and Firebase as the back-end service.



LAUNKLAB UI/UX (2021)

This is a UI/UX design of a Laundry mobile app as a part of the Human-Computer Interaction Class. Designed and prototyped using Figma.



TitipYa (2022)

Titipya is a research project/final college assignment that was done by me and my partner Glainhard Damping under the direction of our advisor namely Sir Stenly Adam, S.Kom, M.Sc. Titipya is a shopping service to supermarkets that provide services for customers and also couriers. Developed using the React Native framework and Firebase as a back-end service



Titipya Admin (2022)

This web app is made for the admin of the Titipya App. Created with ReactJS library and connected to the same database as the Titipya Mobile App, that is using Firebase Real-Time Database.



PinCam (2022)

The idea of this app is a photography tools rental. This is the project of the Full-Stack Development Class that I worked on with my partner Claudio Mambu. The application created with ReactJS library for the front-end and Firebase real-time database for the back-end service, implemented CI/CD using GitHub Actions.
